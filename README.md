# 本地图像修补系统 (Wm-Remover)

本仓库提供一套完整的本地化图像修补与背景修复系统，支持通过 Web 界面进行图像上传与自动化像素级修复。

## 🏗️ 系统概述

项目基于 Node.js 与 Python 构建，采用分层架构以实现高效的本地化处理：

- **服务层**: 负责任务分发、图像存储及前端交互。
- **处理层**: 采用图像修补（Inpainting）算法，实现对目标区域的像素重构与纹理融合。

## 📂 目录说明

- `ai-engine/`: 图像处理核心模块。
- `watermark-remover/`: 系统服务端逻辑与静态前端资源。
- `scripts/`: 系统维护及运维脚本。

## 🚀 核心优势

- **私密性**: 所有图像均在本地环境处理。
- **稳定性**: 独立部署运行，不依赖外部第三方服务接口。
- **高性能**: 针对本地硬件优化的算法实现。
