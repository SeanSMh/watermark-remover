# 阿旺·本地 AI 去水印系统 (Wm-Remover)

本仓库包含了经过“破晓行动”重构后的本地去水印全套系统，彻底摆脱了对外部 AI API 的依赖。

## 🏗️ 项目架构
项目采用 **Node.js (指挥官) + Python (本地特工)** 的协同架构。

### 1. 指挥部 (Node.js 后端 - 5004 端口)
- **位置**: `watermark-remover/backend`
- **职责**: 处理前端上传、任务分发、结果存储。

### 2. 特工行动组 (Python AI 引擎 - 5005 端口)
- **位置**: `ai-engine/`
- **核心引擎**: 基于 **Lama (Inpainting)** 的本地化实现。
